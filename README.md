prm.js
======

# Requirements
Node v4.2.0

# How to Start
```shell
> npm install
> node ./src/main/server/app.js 8080 ./src/main/client
<browser> http://localhost:8080/
```

# How to Run One Test
https://facebook.github.io/jest/docs/en/cli.html#testnamepattern-regex
```
> yarn test --testNamePattern=.*linear.*
```
